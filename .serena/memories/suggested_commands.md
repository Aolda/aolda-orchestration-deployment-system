Project commands:
- `cd /Users/ichanju/Desktop/aolda/AODS` to enter repo root.
- `make backend-run` to start backend (`cd backend && go run cmd/server/main.go`).
- `make frontend-run` to start frontend (`cd frontend && npm run dev`).
- `make check` to run lightweight checks (`go vet ./... || true` and `npm run lint || true`).
- `cd frontend && npm run build` to build frontend.
Useful Darwin shell commands: `ls`, `pwd`, `cd`, `rg`, `find`, `git status`, `git diff`.
